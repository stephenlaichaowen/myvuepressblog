# 物聯網浪潮來襲

IOT (Internet Of Things) 中文譯成物連網, 是繼大數據後最夯最熱的科技趨勢. 所謂物連網的意思是萬物皆能連上網路, 比如燈跑 (本文即將談到), 門窗, 冷氣, 冰箱, 微波爐, 風扇, 甚至衣物等靠著智慧晶片 (本文即將談到的 ESP8266 Wifi 模塊) 連上網路, 使用者再透過電腦, 行動裝置控制及接收資料, 讓生活更加便利, 使得人與物之間再也沒有距離.

我們可以在家裡的螢幕前遙控遠在美國的無人機

醫師可以在家進行遠端診療

回家路上透過手機先行打開冷氣

出遠門透過行動裝置遙控機器餵食寵物

這是一種革命性的突破, 讓人類與科技更緊密結合, 這個以前在科幻電影才有的概念, 如今正全面襲來, 所有人正全面接受物連網浪潮帶來的便利及改變,

# ESP8266 WiFi 模塊系列

ESP8266 WiFi 模塊是深圳市安信可科技有限公司開發出來的, 它的特色 ~

  • 體積小 : 只有拇指頭的大小

    • 由於體積真的非常迷你, 非常容易和物件連接, 只需一顆電池, 簡單的焊接就可以賦予物件連網的能力.

  • 價格低廉

    • 只賣 60 ~ 100元台幣, 人人皆負擔的起, 對學生而言更是福音, 用少許的錢就能輕鬆製作專題, 將創意想法轉變成真.

  • 開發容易

    • 開發者可以透過 Arduino 友善的操作介面輕鬆的編寫程序並下載到模塊上. 安信可公司提供 豐富的程序範例供用戶學習學習.
