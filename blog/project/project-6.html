# 前言

陣列是 JavaScript 中很重要的課題, 熟悉陣列的操作會讓開發變得輕鬆愉快. 今天六角學院臉書官網有人貼了這個教學 - 用一張圖把所有陣列的操作清楚說明, 很讓人驚艷. 必須要推! 發表者在 Youtube 頻道發佈了教學影片, 跟著影片教學打程式碼幫我再度重新溫習陣列的觀念及使用時機和方法, 幫助加深印象,我覺得挺好的. 因為技術只要一段時間沒有接觸就會忘記, 所以透過寫網誌把重點及學習心得用自己的話記錄下來真的很重要. 尤其新的技術不斷冒出, 舊的技術不斷被改良, 變動非常頻繁, 學習方式也在不斷進化改良. 生活每天都充滿驚喜呢!
