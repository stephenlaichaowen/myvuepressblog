    # " 我參加程式集訓三個月後應徵工作, 這是我學到的 "

    # 前言

    我追蹤的 Freecodecamp 臉書頻道出現了一則有趣的貼文, 標題是 "I spent 3 months applying to jobs after a coding bootcamp. Here’s what I learned.", 翻成中文意思是 "我參加程式集訓三個月後應徵工作, 這是我學到的". 作者參加集訓課程三個月後就去應徵工作, 他把學到的用 8 行 javascript 程式碼表示 :

    functrion getDevJob(studying, hardWork, luck) {
  var isPrepared = studying && hardWork && luck
  if (isPrepared) {
    return true
  } else {
    return
  }
}

    # 程式碼解析

    getDevJob 函式帶了 3 個參數分別是 studying, hardWork 和 luck, isPrepared 變數等於 studying && hardwork && luck, 如果 isPrepared 的條件被滿足 (也就是 studying, hardWork, luck 同時成立), 回傳真值, 否則回傳假值. 白話文的意思是 : 想要取得一個網頁開發工作需要學習, 努力和運氣, 如果這 3 樣條件都具備, 那就是準備好了, 如果缺少其中一項, 就表示還不具備找到工作的條件. 下面有一位網友留言 :

    const getDevJob = (studying, hardWork, luck) => studying && hardWork && luck

    他用一行代碼就解決了. 很多人看了都哈哈大笑. 這個梗就是貼文者根本就還沒有準備好, 別人可以用 1 行代碼他卻需要寫到 8 行, 所以表示他 javascript 的造詣還很淺, 想找到一個程序開發的工作應該挺困難的吧! 😆😆😆
