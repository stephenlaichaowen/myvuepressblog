# 使用 Bootstrap 4 + Vue 建立一個教學網站

# 前言

這門課是六角學院推出的課程, 六角學院是最近崛起的網路教學平台, 創辦人為廖洧杰及王志誠, 兩位都是資深的網頁前端工程師, 推出的課程在台灣非常受歡迎. 我個人也長期追蹤六角學院的官網及 Youtube 頻道, 也購買了幾門課程. 基本上我都給予 5 顆星評價, 不過這門課程我給差評, 我原本以為會有詳細的講解說明, 不過聽了前面幾個單元讓我很失望, 因為都沒有甚麼樣的解釋, 基本上這是一門進階課程, 適合已經有 VueJS 基礎的人. 不過這門課程打的名號是從基礎教起, 所以反差很大, 這門課的講師王志誠明顯教學經驗不足, 和 Hiskio 的姚煒揚教學功力高下立判, 不過也不是沒有可取之處啦, 這門課全部教學都是搭配 Bootstrap 4 進行, 所以可以綜觀的檢視 VueJS + UI 框架的整合應用. 這門課我只聽了前面一小部分, 因為我真的聽不下去, 講師教學的經驗和功力真的影響學習的意願.
