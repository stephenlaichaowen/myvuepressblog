# 使用 Framework7 和 VueJS 建立一個行動裝置 APP

# 前言

充滿驚喜的 Youtube 頻道今天出現了一個視頻, 主題是 "Building Mobile Apps with Vue and Framework 7 - Simon MacDonald", 這是一個技術演講場合, 講員我猜應該是 VueJS 開發團隊的成員, 因為他提到了 Sarah Drasner 和 Jean Looper 這兩位很活躍的 Vue / NativeScript 開發者和推廣者. 其實我很早以前就曉得 framework7 這個框架, 不過因為當時不懂前端框架, 所以也就沒有深入地去研究及實作. 不過對於精心設計的組件很感興趣. 現在學會了 Vue, 當然要來嘗試一下這個工具.

Framework7 is an open source HTML framework to develop apps with iOS & Android native look and feel

翻成中文的意思是: framework7 是一個用來開發 Android 和 iOS 行動裝置 App 的開源 HTML 框架. "native look and feel" 意思是看起來和感覺上像是原生的 App, 也就是說 Framework7 的組件和 Android / iOS 的組件長的幾乎一模一樣, 使用者很難分辨出到底 App 用的是 native component 還是 Web compoent. 對於使用者來說應用程序只要好用, 介面親民易上手, 這樣其實就夠了, 至於效能雖然有差別,不過由於現在行動中置硬體技術越來越強大, 讓 native, compile, hybrid, web app 的界線越來越模糊了.
