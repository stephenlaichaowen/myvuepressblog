    # 免費的網站託管 / Free Web Hosting

    目前有許多大型的 IT 公司提供免費的網路託管服務如知名的 Github, Heroku, Netlify, Firebase 等, 提供從 500MB ~ 1G 不等的免費空間及託管服務. 我所用的網路託管叫做 Glitch, 不在名單內, 不過我很喜歡, 它的特色: 上傳檔案方便: 在檔案總管用拖拉的方式就可以輕鬆上傳檔案到雲端; 可以在雲端編程: 這點是我最喜歡的, 我不需要在本機建立一個專案, 用開發環境編寫程式, 可以直接在服務器上寫 code 真的很便利. 而且還支援導入 Github 專案呢! Glitch 最強大的一點是支援 nodejs 全端開發, 所以我們可以無痛開發 MEAN, MERN, MEVN 等全端產品, 非常強大又貼心的功能. 我大推!!!


    Netlify 是最近很夯的網站託管服務, 特色是手續簡便, 能夠非常快速的將專案部署到雲端, 所以很受歡迎, 不過有一個大問題, 就是不允許移除專案, 如果我們不想要某個網站, Netlify 沒有提供移除的功能, 所以個人網站可能很多很雜, 這點真的滿落後的, 不過 Netlify 支持部分後端, 讓它的評價維持平盤.


    Github 的靜態網頁託管一直很受歡迎而且非常多人使用. 把 master 設定 gh-pages 就可以把專案變成網站, 不再只是冷冰冰的檔案. 不過只支持前端網頁, 如果想要加上後端則是不支援的.


    Heroku 是另一個強大的網站託管服務, 支持全端網站上傳, 缺點是設定稍嫌複雜, 手續繁多, 不像 Netlify, 幾個按鍵就完成網站部署.


    Firebase 我沒試過, 所以暫時無法評論.
