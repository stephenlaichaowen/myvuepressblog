# Grid layout VS Bootstrap

  • We can change the order of the grid in RWD with media query, but we achieve this with bootstrap

  • Grid drastically reduce the amount of codes when compared to that of bootstrap


# Chapter 2 : Your first grid

step 1 : turn container into grid

  • display: grid

step 2 : decide column width : grid-template-columns: auto auto 50px

  • column 1 : responsive

  • cloumn 2 : responsive

  • column 3 : 50px

step 3: decide row height : grid-template-rows: 50px 50px

  • row 1 : 50px

  • row 2 : 50px

step 4. create an 3px-gap between each child element

  • grid-gap: 3px

# Chapter 3 : Fraction units and repeat

Example : 3 columns, each column takes equal fraction of the container width

  • grid-template-columns: 1fr, 1fr, 1fr or grid-template-columns: repeat(3, 1fr)

Example : 2 rows, each row is 50px height

  • grid-template-rows: 50px 50px or grid-template-rows: repeat(2, 50px)

shorthand expression : grid-template: rows / columns

  • grid-template: repeat(3, 1fr) / repeat(2, 50px)

# Chapter 4 : Positioning items

# 12 column layout

Example 1

maket 12-column each one with equal fraction

  • grid-template-columns: repeat(12, 1fr)

make 3 rows each one with responsive height

  • grid-template-rows: repeat(3, auto)

header and footer takes full width of the container

  • grid-column-start: 1 + grid-column-end: 13

  • grid-column: 1 / span 12

  • grid-column: 1 / 13

  • grid-column: 1 / -1


menu takes 4 fraction

  • grid-column: 1 / 4

  • grid-column: 1 / span 3

  • grid-column-start: 1 + grid-column-end: 4


Example 2

hide menu and make content full width at breakpoint 720px

media query : max-width: 720px

menu : display: none

content : grid-column: 1 / -1

# Chapter 6 : Auto-fit and minmax

Example 1

fit the container with div (100px) as far as possible

  • grid-template-columns: repeat(auto-fit, 100px)

Example 2

make each column at least 200px wide and responsive to fit the container

  • grid-template-column: repeat(auto-fit, minmax(200px, 1ft))

TIP

with auto-fit and minmax there is need to use media query to achieve RWD
