# 關於 vue-cli

~ 什麼是單一檔案組件 ~

  • 就是 html 模板, css, javascript 都放在同一個檔案裡

  • 一個檔案就是一個組件.

  • 為了要用這個方式開發需要把檔案經過轉換, 打包才能讓瀏覽器順利執行程式.

  • 我們透過執行 vue-cli 命令編譯單一檔案組件成瀏覽器看得懂的格式.


~ 安裝 vue-cli ~

  • 終端機 : npm i - g vue-cli

# import 與 export

現代前端開發會習慣性把程式拆開成很多檔案, 在 Vue 的開發一個檔案代表一個組件.

~ Counter.js ~

const Counter = Vue.component('counter', {
  // ...
})

export default Counter

import Counter from './Counter'

new Vue({
  el: '#app',
  components: {
    Counter
