# 表單設計 : 文字欄位並非只有 "text"

設計行動裝置的輸入欄位類型

  • 點擊 郵件 輸入框出現 英文鍵盤 email: <input type="email">
  • 點擊 電話 輸入框出現 數字鍵盤 tel: <input type="tel">
  • 點擊 數量 輸入框出現 符號鍵盤 email: <input type="number">

# 基礎篇 : 響應式圖片設計

img { width: 100% } :

  • 父元素多寬圖片就多寬
  • 父元素太寬圖片會糊掉

img { max-width: 100% }

  • 保留原圖的大小
  • 實務上常用的方式
  • 父元素縮小圖片隨著縮小

# 基礎篇 : 響應式圖片 reset 教學

  max-width: 100%;
