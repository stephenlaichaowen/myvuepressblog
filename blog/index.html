# 寫部落格的初衷

我其實很早就有寫部落格的想法和念頭, 不過那時對網頁開發並不熟悉, 沒有深入鑽研相關技術, 所以一直懸而未決. 直到近兩年學習網頁開發才慢慢接觸. 我前端學完 Vanilla Javascript / Jquery 之後, 就學框架, 我選擇的是 VueJS, 因為 VueJS 是大陸的尤雨溪大大獨立開發的, 現在有一個很強大的團隊在做開發和維護. 台灣的線上教學也傾向 VueJS, 因為有非常完備的文檔資料/社群, 而且有中文版的官文文件, 非常適合華人學習.

無意間在 youtube 頻道發現一個以 VueJS 為基礎專為寫部落格開發的框架 ~ vuepress 介紹影片, 依照指示一步一步在短時間就建立一個部落格, 雖然版面很單調, 但還是很讓我驚豔. 因為我可以用簡易上手的 markdown 語言編寫文檔內容, 快速的產出文件, 真是非常棒的一件事 !!!

# 寫部落格的好處

  • 記錄生活的點點滴滴

    • 其實寫部落格就像寫日記, 以前在學校老師要求每天都要寫日記, 那樣的日子說實在還滿令人懷念, 寫日記可以培養作文的能力, 增進中文造詣, 是非常好的一件事. 最近幾年網路技術進步神速, 所有的作業都移到網路了, 敲敲鍵盤, 划划手機就可以產出一篇文章, 真是輕鬆又寫意.

  • 和網友意見交流

    • 把部落格發佈到雲端讓全世界每一個人閱讀, 然後留言發表評論何感想, 可以從中了解來自四面八方網友的想法, 互相交流學習, 透過分享快速結交認識朋友, 是網路時代才辦得到的事.

  • 讓寫部落格成為全民運動吧

    • 透過寫部落格輸出自己的知識, 再回過頭來看, 是一件很有成就感的事情, 堅持下去, 世界能因為我們每一個人變得有一點點不一樣
